About

This application is helping you to get an overview on the status of the SaaS services you rely on. It shows you which services are having a maintainance or an incident during the last 15min. Status are updated every 15min by backend running on lzone.de. The backend is a simple old-school feed aggregation script that you can find under https://github.com/lwindolf/multi-status.

This application can be used...

  • on a smartphone to quickly check all service you are interested in.
  • on a desktop (e.g. installed as a local app) serving as a monitoring window for SaaS status.
In any case you might want to configure a filter so only apps you are interested in are shown. To do so click "Filter" in the main screen.
